## Instalação

Você precisará clonar este repositório:

```sh
  $ git clone https://github.com/Bruno-Goudric/escola-magia.git
```

Agora você precisa instalar os pacotes:

```sh
  $ yarn install
```

```sh
  $ npm install
```
